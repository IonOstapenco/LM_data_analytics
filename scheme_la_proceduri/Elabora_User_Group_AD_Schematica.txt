# 🟩 **2. PRELUCRARE – LOGICA PROCEDURII**

## **A. Încărcare grupuri STD / PRO**
```
Lista gruppi Office.txt
     │
     ▼
[Split după separator]
     │
     ├── dacă tip = "std" → GROUP_STD
     └── altfel → GROUP_PRO
```

---

## **B. Căutare fișiere AD**
```
cm.list_files_scandir → găsește toate AD-usersAndGroupsResult-*.csv
```

---

## **C. Procesarea fiecărui fișier AD**
```
┌──────────────────────────────────────────────┐
│ Pentru fiecare fișier AD găsit               │
└──────────────────────────────────────────────┘
                 │
                 ▼
[Citire linii + eliminare header]
                 │
                 ▼
[Curățare linie: \x00, newline]
                 │
                 ▼
[Split după separator → câmpuri y[]]
                 │
                 ▼
[Gestionare linii incomplete (<7 câmpuri)]
                 │
                 ▼
[Excluderi]
   ├─ SamAccountName conține: cr, cre
   └─ GroupName conține: 099, ctx_033, ctx_870
                 │
                 ▼
[Ignorăm dacă y[6] == "Group"]
                 │
                 ▼
[Clasificare]
   ├─ dacă GroupName ∈ GROUP_STD → USER_GROUP_STD
   └─ dacă GroupName ∈ GROUP_PRO → USER_GROUP_PRO
```
